Вертикальная группа чекбоксов
```
<CheckBoxGroup>
    <CheckBox
        text='Текущий счет'
    />
    <CheckBox
        text='Основной счет'
    />
    <CheckBox
        text='Семейный счет'
    />
    <CheckBox
        text='Зарплатный счет'
    />
</CheckBoxGroup>
```

Горизонтальная группа чекбоксов, состоящая из обычных кнопок
```
<CheckBoxGroup type='button'>
    <CheckBox
        disabled={ true }
        text='Текущий'
        type='button'
    />
    <CheckBox
        text='Основной'
        type='button'
    />
    <CheckBox
        text='Семейный'
        disabled={ true }
        type='button'
    />
    <CheckBox
        text='Зарплатный'
        type='button'
    />
</CheckBoxGroup>
```
