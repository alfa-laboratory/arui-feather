        text='Чекбокс'
    />

Горизонтальная группа чекбоксов, состоящая из обычных кнопок
```
<CheckBoxGroup type='button'>
    <CheckBox
        disabled={ true }
        text='Чекбокс'
        type='button'
    />
    <CheckBox
        text='Чекбокс'
        type='button'
    />
    <CheckBox
        text='Чекбокс'
        disabled={ true }
        type='button'
    />
    <CheckBox
        text='Чекбокс'
        type='button'
    />
</CheckBoxGroup>
```
