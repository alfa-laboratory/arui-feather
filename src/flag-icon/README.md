Режим загрузки через спрайт
```jsx
const countries = require('../lib/countries').default;
const countriesList = countries.getCountries();

<div>
    {
        ['s', 'm', 'l', 'xl'].map(size => (
            <div className='row' style={{ alignItems: 'flex-start', flexWrap: 'wrap' }}>
                {countriesList.map(item => (
                    <FlagIcon
                        country={ item.iso2 }
                        mode='sprite'
                        size={ size }
                    />
                ))}
            </div>
        ))
    }
</div>
```
