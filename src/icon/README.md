```jsx
<div>
```

Пример использования цветных иконок
```jsx
<div>
    {['action-error', 'action-ok'].map(icon => (
        <div className='row' key={ icon }>
            {['s', 'm', 'l', 'xl'].map(size => (
                <div className='column l' key={ size }>
                    <Icon
                        name={ icon }
                        size={ size }
                        colored={ true }
                    />
                </div>
            ))}
        </div>
    ))}
```
