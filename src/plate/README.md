```jsx
<Plate hasCloser={ true }>
    <Heading size='s'>
        Альфа-Мобайл 9.0
    </Heading>
    <Paragraph>
        Новый, превосходный, невероятный, беспрецедентный, большой, внушительный, четкий, безотказный, волшебный, понятный. +40 новых возможностей твоего мобайла!
    </Paragraph>
</Plate>
```
