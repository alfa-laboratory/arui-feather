Вертикальная группа радио инпутов с заголовком
```jsx
<RadioGroup>
    {['Один', 'Два', 'Три', 'Четыре'].map(text => (
        <Radio
            text={ text }
            key={ text }
            value={ text }
        />
    ))}
</RadioGroup>
```

Та же группа, теперь с заголовком и ошибкой
```jsx
<RadioGroup error='Обязательно' label='Сколько?'>
    {['Один', 'Два', 'Три', 'Четыре'].map(text => (
        <Radio
            text={ text }
            key={ text }
            value={ text }
        />
    ))}
</RadioGroup>
```
